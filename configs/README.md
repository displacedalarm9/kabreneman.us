# Node Configurations

This directory contains XML configuration files for various nodes in the personal data management system.

## Files

- `config_NODE-ARC-003_CustomArchival.xml` - Custom archival storage node configuration
- `config_NODE-OPS-001_Legion5Gen10.xml` - Legion 5 Gen 10 operational node
- `config_NODE-VR-002_LegionPro5Gen10.xml` - Legion Pro 5 Gen 10 VR node

## Configuration Format

Each configuration file includes:
- System information (product name, model, node role, CSN)
- Hardware specifications
- Software stack
- Profile overlay settings
- Networking configuration
- Environment tags
