# KABDMSV2 - Personal Data Management System

A file management system for organizing and managing personal data across repositories.

## Overview

KABDMSV2 (KAB Data Management System Version 2) is the **management system/software** component designed to handle file organization, tracking, and workflow management for personal data.

## Repository Structure

This project follows a separation of concerns:

| Repository | Purpose | Visibility |
|------------|---------|------------|
| **KABDMSV2** | File management system, software, and utilities | Public |
| **kabreneman.us** | Actual data files (financial records, logs, personal documents) | Private (recommended) |

## Purpose

This repository contains:
- System configuration and metadata
- Management utilities and scripts (planned)
- Documentation and procedures
- Templates for data organization

## Related Repositories

- [kabreneman.us](https://github.com/displacedalarm9/kabreneman.us) - Data storage repository

## Getting Started

1. Clone this repository for the management system
2. Set up your data repository separately (recommended: keep private)
3. Configure the system to point to your data location
